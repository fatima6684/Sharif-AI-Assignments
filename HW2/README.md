# Regression Models

## Overview

This project explores fundamental regression techniques used in machine learning for predicting continuous values. It includes data preprocessing, model training, performance evaluation, and comparison of different regression approaches.

---

## Features

- Data analysis and preprocessing.
- Implementation of regression algorithms.
- Model training and prediction.
- Performance evaluation using appropriate metrics.
- Visualization of results and model behavior.

---

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## Course

Developed as part of the Artificial Intelligence course.
